# Markdown ➡ テーブル変換くん

CSV・TSV・Markdown表を、**noteに貼りやすいグラフ／表のPNG**へ変換するブラウザツールです。
インストール不要、サーバー不要。HTMLファイルをダブルクリックするだけで動きます。

👉 **[デモを開く](https://aul-dox.github.io/Markdown-to-Table/)**

---

## 主な機能

### データ入力
- Markdown表（`| col | col |` 形式）
- CSV / TSV ファイルアップロード
- スプレッドシートからのコピペ
- セル内 `**太字**` / `*斜体*` 記法に対応

### 出力モード

| モード | 説明 |
|--------|------|
| 棒グラフ | 単系列の縦棒 |
| 折れ線グラフ | トレンド表示 |
| エリアグラフ | 面塗りつぶし |
| 集合棒グラフ | 複数系列を並べて比較 |
| 積み上げ棒グラフ | 構成比の可視化 |
| 散布図 | 2軸の相関 |
| 円グラフ | 割合表示（ドーナツ型） |
| **表** | スタイル付きテーブルをPNG化 |

### カスタマイズ
- 6つのカラーテーマ（Terracotta / Marine / Forest / Mono Ink / Berry / Sunrise）
- ヘッダー背景色・文字色、グラフ主色・副色を個別指定
- 文字サイズ・角丸・凡例・値ラベル・補助線をトグル
- 出力サイズ：横長 1200×675 / 正方形 1080×1080 / 縦長 1080×1350

### エクスポート
- **PNG保存**（html2canvas、解像度 2×）
- **クリップボードへコピー**（そのままnoteに貼り付け可）

---

## 表モード限定の操作

表モードでは、出力後のプレビュー上で直接レイアウトを調整できます。

| 操作 | 方法 |
|------|------|
| 列の並び替え | ヘッダーセルを左右にドラッグ |
| 列幅の変更 | ヘッダー右端（`↔` カーソル）をドラッグ |

> [!NOTE]
> 列操作はプレビュー上のみの変更です。PNG保存・クリップボードコピーにはそのまま反映されます。

---

## 使い方

1. HTMLファイルをブラウザで開く
2. 左パネルにMarkdown表・CSV・TSVを貼り付けるかファイルを選択
3. グラフ種類・テーマ・サイズを選ぶ
4. 「プレビュー更新」を押す
5. 「PNGで保存」または「画像をクリップボードへコピー」
